# 2DOF Robot Arm Documentation

## Package Setup
To set up the robotic arm package, you need to follow these steps:
1. Clone the repository:
   ```bash
   git clone https://github.com/energy6314/2dof_robotarm.git
   cd 2dof_robotarm
   ```

2. Ensure you have Python installed (version 3.6 or later).

3. Install the required packages:
   ```bash
   pip install -r requirements.txt
   ```

## Hardware Requirements
- 2 DOF Robot Arm Kit
- Microcontroller (Arduino, Raspberry Pi, etc.)
- Power supply for the servos
- Jumper wires

## Installation
1. Connect the robot arm servos to the microcontroller according to the provided circuit diagrams.
2. Upload the control code to the microcontroller:
   ```bash
   # For Arduino, use the Arduino IDE to upload the code.
   ```

3. Ensure all hardware is securely connected and powered on.

## Usage Instructions
- To run the robotic arm, execute the following command in the terminal:
  ```bash
  python main.py
  ```
- Follow the prompts to control the robot arm.
